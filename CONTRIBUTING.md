# Contributing to FocusFlow AI

We love your input! We want to make contributing to FocusFlow AI as easy and transparent as possible, whether it's:

- Reporting a bug
- Discussing the current state of the code
- Submitting a fix
- Proposing new features

## Our Development Process

1. **Fork the repo** and create your branch from `main`.
2. If you've added code that should be tested, **add tests**.
3. Ensure the test suite passes.
4. **Issue that pull request!**

## Report Bugs Using GitHub Issues

We track bugs, enhancements, and feature requests using GitHub Issues. Report a bug by opening an issue:

- **Describe the bug**: Explain clearly what happened and what you expected to happen.
- **Provide reproduction steps**: Include code snippets, input values, and terminal commands.
- **Provide environment details**: OS, Python version, and dependency versions.
